Kalkulator Infinity

Ferramenta científica em CLI para Termux/Linux, focada em cálculo avançado, física, cosmologia e análise científica racional.


Instalação

pkg update && pkg upgrade -y
pkg install git bc -y
git clone https://github.com/deadch/Kalkulator-Infinity
cd Kalkulator-Infinity
chmod +x kalkulator_infinity.sh
./kalkulator_infinity.sh


Recursos

Matemática científica de alta precisão

Física clássica e moderna

Cosmologia e limites físicos

Engenharia elétrica (Tesla)

Benchmarks e stress numérico

Análise científica e educacional

Bash
